# П07: Docker Basics

## Описание
Создание Docker образа с веб-сервером Nginx с поддержкой SSL/TLS, настройка проброса портов и HTTPS перенаправления.

## Что сделано
- Написан Dockerfile на основе официального образа nginx
- Создана кастомная индексная страница (index.html)
- Настроена конфигурация nginx с HTTP→HTTPS редиректом
- Сгенерирован самоподписанный SSL сертификат
- Собран Docker образ с тегом `devops/nginx-server`
- Запущен контейнер с пробросом портов 54321→80 и 54322→443
- Проверена работа через HTTP и HTTPS

## Технологии
- Docker (контейнеризация)
- Dockerfile (инструкции сборки образа)
- Nginx (веб-сервер с SSL)
- SSL/TLS (самоподписанные сертификаты)
- Port forwarding (проброс портов)
- Volume mapping (монтирование файлов)

## Структура проекта

- `P07-Docker-Basics/` — корневая папка лабораторной работы
  - `nginx/` — папка с файлами для Docker образа
    - `Dockerfile` — инструкции для сборки Docker образа
    - `index.html` — кастомная индексная страница ("DevOps Course 2026!")
    - `nginx-conf/nginx.conf` — конфигурация nginx с HTTP→HTTPS редиректом и SSL
  - `ssl/` — папка с SSL сертификатами
    - `nginx.crt` — публичный сертификат (самоподписанный)
    - `nginx.key` — приватный ключ RSA 2048 бит
  - `deploy.sh` — скрипт для автоматической генерации SSL, сборки и запуска контейнера
  - `README.md` — документация проекта
  - `screens/` — папка со скриншотами выполнения
