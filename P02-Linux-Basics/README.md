# П02: Linux Basics - Автоматизация в Linux

## Описание
Создание Bash-скрипта для получения данных о погоде через API open-meteo.com, 
настройка автоматического выполнения через cron, вывод результатов через веб-сервер nginx.

## Цель работы
Написать bash-скрипт, который:
- Принимает город как параметр
- Получает температуру и влажность через API
- Выводит данные в формате "температура" "влажность" дата
- Запускается по cron каждую минуту
- Сохраняет результат в index.html nginx

## Что сделано
- Установлен веб-сервер nginx
- Установлена утилита jq для работы с JSON
- Написан Bash-скрипт weather.sh
- Настроены права доступа к /var/www/html/index.html
- Настроен cron для выполнения скрипта каждую минуту
- Настроен проброс порта в VirtualBox (8080 → 80)
- Проверена работа через curl и браузер

## Технологии
- Bash scripting
- curl (HTTP запросы к API)
- jq (парсинг JSON)
- cron (автоматизация задач)
- nginx (веб-сервер)
- VirtualBox port forwarding
- open-meteo.com API
