# П04: Ansible Basics

## Описание
Развёртывание веб-сервера Nginx на удалённой машине (VM-2: 10.0.2.16) с помощью Ansible playbook.

## Цель работы
- Создать вторую виртуальную машину (клонирование VM-1)
- Настроить сеть NAT (10.0.2.0/24)
- Создать пользователя runner с sudo правами
- Настроить SSH ключи для подключения без пароля
- Установить Ansible на VM-1
- Написать playbook для установки nginx
- Выполнить playbook и развернуть веб-сервер

## Что сделано
- Создана вторая виртуальная машина (клонирование с генерацией новых MAC-адресов)
- Настроена сеть NAT в VirtualBox
- Обе VM переключены в сеть NAT с адресацией 10.0.2.0/24
- Заданы статические IP адреса (VM-1: 10.0.2.15, VM-2: 10.0.2.16)
- Настроен проброс портов (22 на обе VM, 80 на VM-2)
- Создан пользователь runner с правами sudo без пароля
- Сгенерирован и скопирован SSH ключ
- Установлен Ansible на VM-1
- Написан playbook для установки nginx
- Создан inventory файл с описанием целевой машины
- Выполнен playbook и развёрнут веб-сервер
- Проверена работа через curl и браузер

## Технологии
- Ansible (система управления конфигурациями)
- Nginx (веб-сервер)
- SSH (беспарольный доступ)
- YAML (формат playbook)
- VirtualBox NAT Network

## Структура проекта
- `P04-Ansible-Basics/` — корневая папка лабы
  - inventory.ini` — описание хостов
  - playbook.yml` — playbook для установки nginx
  - index.html` — шаблон статической страницы
  - `README.md` — документация
  - `screens/` — папка со скриншотами выполнения
