# П06: Ansible Roles + CI/CD

## Описание
Создание Ansible роли для развёртывания Nginx с виртуальными хостами. 
Настройка GitHub Actions для автоматической проверки кода (ansible-lint).

## Что сделано
- Создана структура роли nginx (tasks, handlers, templates, vars, defaults)
- Реализована установка и настройка Nginx
- Настроены 3 виртуальных хоста через Jinja2 шаблоны
- Выполнен playbook на целевой машине (10.0.2.16)
- Проверена работа всех виртуальных хостов

## Технологии
- Ansible Roles
- Jinja2 Templates
- Nginx (веб-сервер с виртуальными хостами)

## Структура проекта
- `P06-Ansible-Roles/` — корневая папка лабораторной работы
  - `inventory.ini` — описание хостов (целевая машина 10.0.2.16)
  - `playbook.yml` — главный playbook с подключением роли nginx и переменными sites
  - `README.md` — документация
  - `roles/nginx/` — Ansible роль для развёртывания и настройки Nginx
    - `defaults/main.yml` — переменные по умолчанию для роли (список сайтов `sites` со значениями `mehmat.ru`, `fizfak.ru`, `etis.com`)
    - `handlers/main.yml` — обработчики, которые выполняются при изменении конфигов
    - `tasks/main.yml` — основные задачи: установка nginx, создание конфигов, копирование шаблонов
    - `templates/` — Jinja2 шаблоны для динамической генерации файлов
      - `vhost.conf.j2` — шаблон конфигурации виртуального хоста
      - `index.html.j2` — шаблон индексной страницы с приветствием и названием сайта
    - `vars/main.yml` — переменные роли с более высоким приоритетом
  - `.github/workflows/` — папка с конфигурацией GitHub Actions для CI/CD
    - `lint.yml` — workflow для автоматической проверки кода через `ansible-lint`
  - `screens/` — папка со скриншотами выполнения работы
