# П05: Ansible Advanced

## Описание
Продвинутое использование Ansible: сканирование сети NMAP с использованием циклов (loop), 
работа с переменными, вывод результатов через debug модуль, сканирование из файла.

## Цель работы
- Установить NMAP на целевую машину через Ansible
- Просканировать список целей двумя способами:
  1. Через loop по списку из targets.txt
  2. Через файл targets_2.txt с использованием флага -iL
- Использовать debug модуль для вывода результатов
- Обеспечить идемпотентность выполнения

## Что сделано
- Создан inventory файл с описанием целевой машины (10.0.2.16)
- Написан playbook.yml со сканированием через loop
- Написан playbook_2.yml со сканированием из файла
- Использован модуль command для выполнения nmap
- Результаты выводятся через debug модуль
- Обеспечена идемпотентность (повторный запуск не меняет состояние)

## Технологии
- Ansible (система управления конфигурациями)
- NMAP (сканер портов)
- Jinja2 (переменные, шаблонизация)
- Loops (циклы в Ansible)
- Debug модуль (отладка и вывод)

## Структура проекта
- `P05-Ansible-Advanced/` — корневая папка лабы
  - `inventory.ini` — описание хостов 
  - `playbook.yml` — playbook со сканированием через loop
  - `playbook_2.yml` — playbook со сканированием из файла
  - `targets.txt` — список целей для loop
  - `README.md` — документация
  - `screenshots/` — папка со скриншотами выполнения
